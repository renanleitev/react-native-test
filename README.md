# React Native Test

Repositório para testes em React Native.

## Instalando as dependências

    npm i

## Iniciando o servidor

    npx expo start

## Rodando em WSL

    npx expo start --tunnel

## Conectando ao celular

Baixar o aplicativo Expo GO e escanear o QR Code que está no servidor.
